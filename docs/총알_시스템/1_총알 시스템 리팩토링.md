## **총알 시스템 리팩토링 작업**

### **1. 작업 개요**

- **목표**: 총알 시스템을 확장성과 유지보수성을 고려하여 리팩토링.
- **핵심 변경 사항**:
    1. **Resource 기반 관리**: 총알 속성을 `BulletData` 리소스로 관리.
    2. **공통 기능 분리**: 모든 총알이 공통적으로 가질 기능을 `BaseBullet` 클래스에 정의.
    3. **최초 총알 구현**: `BaseBullet`을 확장한 `BasicBullet` 씬 생성.
    4. **발사 로직 통합**: `BulletManager`를 삭제하고, `ShooterComponent`를 도입하여 발사 로직을 중앙화.
    5. **플레이어 스크립트 수정**: 플레이어가 `ShooterComponent`를 통해 총알을 발사하도록 변경.

---

### **2. 주요 변경 사항**

### **2.1. `BulletData` 리소스**

- **파일 경로**: bullet_data.gd
- **역할**: 모든 총알이 공통적으로 가질 속성을 정의.
- **속성**:
    - `speed`: 총알 속도.
    - `max_distance`: 총알이 이동할 수 있는 최대 거리.
    - `damage`: 총알의 데미지.
    - `penetration`: 총알의 관통력(몇 명의 적을 관통할 수 있는지).
    - `bullet_scene`: 총알의 씬(PackedScene).

```python
extends Resource
class_name BulletData

@export var speed: int = 1500
@export var max_distance: float = 700.0
@export var damage: int = 10
@export var penetration: int = 1
@export var bullet_scene: PackedScene

```

---

### **2.2. `BaseBullet` 클래스**

- **파일 경로**: base_bullet.gd
- **역할**: 모든 총알이 공통적으로 가질 기능(이동, 충돌 처리, 관통 처리 등)을 정의.
- **주요 기능**:
    - 총알 이동(`_process`).
    - 최대 거리 초과 시 제거.
    - 충돌 시 관통 처리(`on_hit`).

```python
extends Node2D
class_name BaseBullet

var bullet_data: BulletData
var direction: Vector2
var initial_position: Vector2
var penetration_count: int = 0

func _ready():
    initial_position = global_position

func _process(delta):
    position += bullet_data.speed * direction * delta
    if position.distance_squared_to(initial_position) > bullet_data.max_distance * bullet_data.max_distance:
        queue_free()

func on_hit(target: Area2D):
    if target is HurtboxComponent:
        penetration_count += 1
        if penetration_count >= bullet_data.penetration:
            queue_free()

```

### **2.4. `ShooterComponent`**

- **파일 경로**: shooter_component.gd
- **역할**: 발사 로직을 중앙화하여 플레이어, NPC, 자동 공격 터렛 등에서 재사용 가능.
- **주요 기능**:
    - 총알 생성 및 발사.
    - 총알의 방향 및 회전 설정.

```python
extends Node
class_name ShooterComponent

@export var bullet_data: BulletData
@export var muzzle: Node2D

func shoot(target_position: Vector2):
    var bullet = bullet_data.bullet_scene.instantiate()
    bullet.global_position = muzzle.global_position
    bullet.direction = (target_position - muzzle.global_position).normalized()
    bullet.rotation = atan2(bullet.direction.y, bullet.direction.x)
    bullet.bullet_data = bullet_data

    var foreground = get_tree().get_first_node_in_group("foreground_layer")
    foreground.add_child(bullet)

```

---

### **2.5. `HitboxComponent`**

- **파일 경로**: hitbox_component.gd
- **역할**: 충돌 처리 및 관통 처리.
- **주요 기능**:
    - 충돌 시 `BaseBullet`의 `on_hit` 호출.

```python
extends Area2D
class_name HitboxComponent

@export var damage: float = 0

func _ready():
    area_entered.connect(on_area_entered)

func on_area_entered(other_area: Area2D):
    if not other_area is HurtboxComponent:
        return
    if owner is BaseBullet:
        owner.on_hit(other_area)

```

### **2.6. `Player` 스크립트**

- **파일 경로**: [player.gd](http://player.gd/)
- **역할**: 플레이어가 `ShooterComponent`를 통해 총알을 발사하도록 수정.
- **주요 변경 사항**:
    - `BulletManager` 의존성을 제거.
    - `ShooterComponent`를 통해 총알 발사.

```python
extends CharacterBody2D

@onready var shooter_component = $ShooterComponent

func shoot_bullet():
    var target_position = get_global_mouse_position()
    shooter_component.shoot(target_position)
    turret_sprite.play("fire")
    can_shoot = false
    shoot_cool_time.start()
    audio_stream_player.play()

```

### **3. 작업 결과**

- **확장성**:
    - 새로운 총알을 추가하려면 `BulletData` 리소스를 생성하고, 필요 시 `BaseBullet`을 상속받아 구현.
    - NPC와 자동 공격 터렛도 `ShooterComponent`를 통해 동일한 발사 로직을 재사용 가능.
- **유지보수성**:
    - 발사 로직이 `ShooterComponent`로 통합되어 중복 코드가 제거됨.
    - 총알 속성과 동작이 분리되어 수정이 용이.
- **재사용성**:
    - `ShooterComponent`를 다양한 발사체 주체에서 재사용 가능.

---

### **4. 향후 계획**

1. **새로운 총알 추가**:
    - 폭발형 총알, 상태 이상 부여 총알 등 고유한 동작을 가진 총알을 추가.
    - `BaseBullet`을 상속받아 구현.
2. **NPC 및 자동 공격 터렛 구현**:
    - NPC와 자동 공격 터렛에 `ShooterComponent`를 추가하여 발사 로직 재사용.
3. **BulletData 확장**:
    - 필요 시 `BulletData`에 추가 속성(예: 폭발 반경, 상태 이상 효과 등)을 정의.
